#### MP3 Player GUI for embedded devices

---

##### CDC AUDIO USB Project

*Version: 0.0.1 beta*

![](img/wnd.png)

<br>

Install dependencies: 

* PyQt5
* pyserial
* audiofile
* No additional driver for /dev/ttyACMx for Linux
* ST VCP USB Driver for Windows

Run application:

    ~$ python3 ./usbcdcaudio.py
